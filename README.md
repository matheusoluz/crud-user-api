# Aplicação para cadastro de usuáros - API

## Pré-requisitos

1. Clonar repositório.
2. Ir para diretório clonado.
3. Instalar Node.
4. Instalar dependências.
    > npm install
5. Criar base de dados no MySQL/MariaDB.
    > user: root
    > password: root
    > command: `Create Database unirede;`
6. Executar API.
  `npm run dev`
